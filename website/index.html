Features

復習まで迷わない記録づくり。

01

ミスをすぐ記録

タイトル、内容、科目、重要度、写真をまとめて保存できます。

02

カレンダーで振り返り

ミスが多かった日をひと目で確認し、学習の偏りに気づけます。

03

科目ごとに整理

科目は自由に追加・編集でき、自分の学習スタイルに合わせられます。

04

復習リスト

重要度や日付から、今見返したいミスを探しやすくします。

05

今日のランダム復習

これまでのミスから毎日3件を自動でピックアップ。0時に更新されます。

For study and work

同じミスは、もうしない。

ミスログは、受験勉強の弱点整理にも、仕事での振り返りにも使える記録アプリです。 失敗をただ残すのではなく、次の行動に変えるためのメモとして使えます。

受験・定期テスト向け

間違えた問題を、得点源に変える。

数学の計算ミス、英語の文法ミス、暗記の抜け漏れ。 「なぜ間違えたか」を科目や重要度と一緒に残して、復習すべきポイントを見失いません。

  • 模試や過去問のミスを写真つきで保存
  • 重要度「高・中・低」で優先順位づけ
  • カレンダーでミスが多い時期を確認

社会人・仕事向け

仕事のミスも、次の改善につなげる。

確認漏れ、連絡ミス、段取りの失敗。 仕事で起きた小さな反省を残しておくことで、同じ場面での再発防止に役立ちます。

  • ミスの原因と次回の対策をメモ
  • 業務カテゴリを科目のように管理
  • 振り返りの習慣づくりに活用

How it works

使い方は、3ステップ。

  1. ミスを追加 問題の写真や気づきをその場で残す。
  2. 重要度をつける 高・中・低で、復習の優先度を決める。
  3. あとで見返す 一覧、カレンダー、復習画面の「今日の3件」から確認する。

ミスログは「できなかった」を責めるためではなく、 次にできるようになるための記録帳です。
