学習の記録を、やさしく続ける

間違いを、次の得点に変える。

ミスログは、テストや演習でのミスを写真・科目・重要度と一緒に残し、 復習すべきタイミングで見返せる学習アプリです。

Features

復習まで迷わない記録づくり。

01

ミスをすぐ記録

タイトル、内容、科目、重要度、写真をまとめて保存できます。

02

カレンダーで振り返り

ミスが多かった日をひと目で確認し、学習の偏りに気づけます。

03

科目ごとに整理

科目は自由に追加・編集でき、自分の学習スタイルに合わせられます。

04

復習リスト

重要度や日付から、今見返したいミスを探しやすくします。

How it works

使い方は、3ステップ。

  1. ミスを追加 問題の写真や気づきをその場で残す。
  2. 重要度をつける 高・中・低で、復習の優先度を決める。
  3. あとで見返す 一覧、カレンダー、復習画面から確認する。

ミスログは「できなかった」を責めるためではなく、 次にできるようになるための記録帳です。
